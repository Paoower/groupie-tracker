# Groupie Tracker

This Go program consists on receiving a given API and manipulate the data contained in it, in order to create a site, displaying the information.

## Arborescence

- `main.go`
- `templates/`
    - `500.html`
    - `400.html`
    - `404.html`
    - `artist.html`
    - `index.html`
- `json/`
    - `artists.json`	
    - `dates.json`	
    - `locations.json`	
    - `relation.json`
- `datatypes/`
    - `structs.go`
- `utils/`
    - `artists.go`
## Usage

To clone the repository:
```bash
git clone https://zone01normandie.org/git/faoudia/groupie-tracker.git
```

To run the Program:
```bash
cd groupie-tracker

go run . 
```
